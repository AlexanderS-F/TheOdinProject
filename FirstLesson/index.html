I think I need to create a repository on the GitHub website and clone it to my local work environment using the below command before starting to work on it:

  • git@github.com:AlexanderS-F/TheOdinProject.git - just an example (the filepath will be different for each repo)

I then navigate to the directory using the terminal and create the file I want to use with:

  • touch filename.extension

I can then open the file I want to edit in VSCode and in Google Chrome (if website):

  1. code filename
  2. google-chrome filename

Once I have edited it and wish to stage it, I enter the following code:

  • git add filename

I can then commit it using the below code:

  • git commit

I have it set up to open new file in VSCode where I can enter commit message.

I can then upload my work to the GitHub repository:

  • git push
